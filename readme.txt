Данный репозиторий содержит скрипты импорта для автоматизированной системы сбора и анализа данных

ftp_import.py - импорт с ftp-сервера localhost, порт 2001, cronuser cronuser-pass
wordstat_import.py - импорт отчёта с сервиса wordstat
mysql_import.py - импорт с mysql-сервера localhost, root cronuser-pass

root - расписание crontab
